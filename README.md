Userscript for Greasemonkey/Tampermonkey that allows you to generate images using any local image generator, should work with anything that goes through web requests, ComfyUI, Stable Diffusion, etc, but I've only tested with ComfyUI using Flux.
Incredibly difficult for consistent generations considering the variety of Pax Historia scenarios and gameplay, but, depending on your model, might be worth for a bit of extra immersion.

Pax Historia will have a new option at the top left that says "IMAGE GENERATION | ComfyUI (or) Stable-Diffusion," tap on that, adjust your settings accordingly.
If using ComfyUI, you can upload your workflow here. Images will generate after every action. You can tap the image for a full-sized preview.
I invite anyone with actual coding experience, consider this as a technical demonstration, a proof of concept, there's definitely a better way to do this, but, this is the best way GPT has been able to come up with for me.
Enjoy!
