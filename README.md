# Pax Historia Image Generator

Userscript for Greasemonkey/Tampermonkey that allows you to generate images using any local image generator, should work with anything that goes through web requests, ComfyUI, Stable Diffusion, etc, but I've only tested with ComfyUI using Flux.
Incredibly difficult for consistent generations considering the variety of Pax Historia scenarios and gameplay, but, depending on your model, might be worth for a bit of extra immersion.


# How To Use
Pax Historia will have a new option at the top left that says "IMAGE GENERATION | ComfyUI (or) Stable-Diffusion," tap on that, adjust your settings accordingly. If using ComfyUI, you can upload your workflow here. Images will generate after every action. You can tap the image for a full-sized preview.

# Text AI
Most models won't do very well with giant swathes of text, so you can use this to compress it using your own local model, or my recommendation, any fast free model on OpenRouter, most text models should do.

# Workflows
You can upload your ComfyUI workflow here, I've been tampering with Flux, but I'm sure there are plenty of good options out there. Tweak your settings until you find what works for your playthrough. 

#To Developers And Hobbyists

For now, consider this as a technical demonstration or a proof of concept, there's definitely a better way to do this. But, this is the best way GPT has been able to come up with for me for now. Enjoy!
